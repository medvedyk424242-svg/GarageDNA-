# GarageDNA

**Формальные, машиночитаемые языки описания музыки для AI-агентов.**  
Проект MashUp Lab, финал сессии 2026-02-12.

## Философия

- **Образ, а не параметры** — музыка описывается через метафоры и архетипы.
- **Протокол «Дирижёр»** — база (70%) + акцент (30%).
- **Бас = гравитация**, ритм = нервная система, энергия ≠ давление.

## 8 таблиц-языков

| Таблица | Назначение |
|--------|------------|
| TABLE_Q | Bass Control Standard — грамматика баса |
| TABLE_R | Groove Interaction Matrix — отношения Bass/Kick/Hi-Hats |
| TABLE_S | Clash & Masking Ruleset — разрешение конфликтов |
| TABLE_U | Pressure Architecture — 6 состояний давления |
| TABLE_V | Energy Engine — психоакустическая энергия |
| TABLE_W | Genre Bass DNA — жанровые эталоны баса |
| TABLE_X | Rhythm DNA Engine — грамматика ритма |
| TABLE_J | DNA Segment — унифицированный формат сегмента |
